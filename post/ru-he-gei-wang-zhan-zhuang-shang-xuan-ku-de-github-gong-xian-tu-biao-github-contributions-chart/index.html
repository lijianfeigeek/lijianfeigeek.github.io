如何给网站装上炫酷的 Github 贡献图表(Github Contributions Chart)

2024-12-22 3 min read # 如何做 # Web

程序员的“朋友圈”——Github 贡献图标，你拥有了吗？

提到程序员，我们脑海中总是浮现出一个个 hunched over the keyboard, 手指飞快敲击键盘的身影。而想要证明自己不是只会打字的码农， Github 贡献图标就是最好的“朋友圈”展示！它就像程序员的日历，记录着每一个 commit、每一个 push，展现着我们辛勤耕耘的成果。

但是，仅仅局限于 Github 的个人页面展示，未免有些小家子气，想要在自己的网站或博客上炫耀一下这个“战绩”？别急，今天就来教大家如何给网站装上一个闪瞎人眼的 Github 贡献图表！

Githubchart-api：让你的贡献图“活”起来！

想要实现这个功能，我们需要借助强大的 Githubchart-api。它就像一个神奇的魔法棒，能够将静态的 Github 贡献数据转化为动态、可视化的图表。官网和 Github 仓库地址都可以在网上轻松找到，详细的使用说明也一应俱全。

  • Github Chart API
  • 2016rshah/githubchart-api: 📅 Embed github contributions chart as image
我的代码贡献记录就用 Githubchart-api 生成的，要知道这可不是一张简单的图片，而是实时更新的！每当你敲下一行代码，提交一次更改，图表都会随之变化，展现出你不断进取的精神面貌。

使用方法：简单易懂，上手即刻成为“炫耀狂魔”！

在你的网页中插入一段简单的代码，就能轻松实现 Github 贡献图表的展示。

<img src="https://ghchart.rshah.org/github-username" />

将 github-username 替换成你的 Github 用户名即可，是不是超级简单？

自定义颜色：打造个性化的“代码炫耀”风格！

默认的绿色方格虽然经典耐看，但也难免有些单调乏味。Githubchart-api 还支持自定义颜色，让你打造独一无二的个性化图表！

例如，如果你想要一个基于十六进制颜色的粉色主题图表 #EA4C89，只需要在标签中加入颜色的十六进制代码即可：

<img src="https://ghchart.rshah.org/EA4C89/github-username" />

是不是瞬间感觉更加“潮”了呢？

Github 贡献图表不仅仅是“炫耀”，更是自我激励的工具！

Github 贡献图表，不仅仅是一个炫耀个性的工具，更可以成为自我激励的动力源泉。每天看着图表上不断累积的贡献，你就会更有动力去学习、去实践，去挑战自己。

当你打开网站，看到自己精心打造的代码贡献图表时，那种成就感和自豪感油然而生！这将是你坚持 coding 路上的强大动力。
