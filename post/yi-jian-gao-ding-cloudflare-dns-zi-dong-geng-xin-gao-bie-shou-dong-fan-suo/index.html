DDNS-GO + Cloudflare 解决动态 IP 变化

2023-09-02 1 min read # Cloudflare # NAS

手动更新Cloudflare DNS太麻烦？担心IP变动后还得手动改？这里有个一站式解决方案！👇

DDNS-GO

使用 DDNS-GO 自动获得你的公网 IPv4 或 IPv6 地址，并解析到对应的域名服务。

1. DNS 服务商配置

1.1 创建 API 令牌

1.2 编辑区域DNS

1.3 选择区域资源（选择你的域名）

1.4 将生成的令牌填写

1.5 保存

2. 域名解析

  1. 域名解析选择类型A；
  2. 解析地址随便填写一个IP；
  3. 代理状态关闭；

3. 查看日志

4. 检查域名解析


可以看到 IP 被自动解析到了 Cloudflare 。
