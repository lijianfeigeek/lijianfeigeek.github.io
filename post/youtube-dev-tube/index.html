YouTube 优质 Dev 频道推荐

2022-06-18 13 min read # 好视频

Eli the Computer Guy 该频道涵盖了从Web开发到网络等各种主题，是YouTube上每个技术领域的顶级学习资源。频道有大量关于网络编程的视频，并配有现场实际演示。

TheNewBoston 拥有超过200万订阅者，是最受欢迎的学习编程语言的网络开发YouTube频道。该频道由Bucky Roberts创立，是许多初学者免费网络学习的热门来源。它有大量的后端和前端语言教程，适用于新手和高级学习者。

Derek Banas 为初学者和高级开发人员带来了关于不同编程语言的广泛网络教程。已经上传了1000多个视频，这个网络开发YouTube频道为每种语言都有单独的播放列表，并且每周定期更新两次。

programming knowledge 频道提供了大量关于顶级后端/前端编程语言的视频教程，包括Python、Swift、PHP、Ruby 等。它是学习Web开发的最佳YouTube频道之一，因为它有关于其他所有框架的详细视频，并且每天有成千上万的访问者。

Edureka 对于绝对初学者来说，Edureka 是学习Web开发的最佳YouTube频道之一，因为它提供了涵盖各种主题的不同语言的大量视频。从DevOps到人工智能 (AI)，从机器学习到Python，该频道提供了有关主要技术的不同视频教程。

Traversy Media 是有关不同编程语言的最新网络教程的绝佳来源。随着订阅者达到725K标记，该频道提供有关主要开发和脚本语言（如 Angular、Node、React.js、Ruby、Python 等）的深入学习教程。

LearnCode.academy 如果想快速学习编程，LearnCode.academy是轻松学习Web开发的最佳YouTube频道之一。它有各种不同的前端/后端语言的视频教程，列出了单独的播放列表以便于分类。

Programming with Mosh Mosh提供了大量教程，不仅包括流行的Web语言，还包括在开发世界中迅速占据一席之地的新兴Web技术。该频道提供长视频（长达数小时），详细讨论语言的每个线程和主题。

DesignCourse 由Gary创立，是一个知名的YouTube频道，用于学习UI/UX世界的最新动态。它有几十个前端开发视频，提供详细的技术知识来学习最新的网页和图形设计方法。

Hitesh Choudhary 拥有探索新语言的诀窍，定期带来各种编程语言的视频，主要是后端。除了上传科技视频外，他还经常在频道上传自己的Vlog，讨论最新的市场趋势。

DevTips 另一个包含多个开发教程的热门频道，提供各种语言的最新每周视频。该频道拥有不断增长的订阅者列表，并上传了大量视频，涵盖了趋势发展主题和优化技巧。

Academind 提供有关前端语言的综合视频教程，包括Angular、Vue、React.js 等。从基础到高级，这个前端编码YouTube频道提供了各种视频指南，即使是初学者也可以帮助他们快速开始使用最喜欢的语言。

LevelUpTuts 由Scott Tolinski和Ben Schaaf于2012年创建，是YouTube上开发人员了解热门开发主题最喜欢的地方之一。该频道拥有1000多个视频，拥有不断增长的订阅者列表，使其成为平台上收视率最高的开发频道之一。

Treehouse 自称是一所在线学校，提供一流的Web开发教程和实际演示。该频道的特点是上传实时编码教程，帮助开发人员通过实时运行时问题答案学习技巧。

LearnWebCode 如果想主要了解 WordPress、CSS/HTML、JavaScripting等，此频道可以让你快速入门。它有大量由大块组成的前端开发视频，详细解释了所有核心脚本主题。

Adam Khoury 总频道浏览量飙升至3000万，也是YouTube上最受关注的开发频道之一，主要是因为其易于理解的编码教程。作为初学者，你可以在频道上找到与JavaScripting、CSS/HTML、PHP等相关的简单教程。

Darrel Wilson 如果有了解WordPress开发最新趋势的诀窍，你应该关注YouTube上Darrel Wilson的知识库。他的频道在WordPress爱好者中很受欢迎，他会定期更新有关该平台的热门新闻和主题，并为使用WordPress构建高级和令人惊叹的网站提供方便的教程。

Codecourse 是轻松学习编码的最佳 YouTube 频道之一，因为它可以让开发人员了解最新的网络语言及其优化技巧。它上传了大量视频，将每个视频分类在单独的播放列表中，让新手找到他们想要的东西。从C#到Laravel和最新的 Nuxt.js，该频道提供了由深入分析组成的广泛教程。

Stefan Mischook 是YouTube上的知名人物，他在自己的频道上创建和上传有关各种开发语言的常规视频。他在每个视频上都有大量的追随者，并且频道订阅者的名单也在不断增长。

Joe Parys Academy 是另一个受欢迎的YouTube频道，您可以在其中获得有关Web开发行业最新动态的大量信息和新闻。这家伙是一个纯粹的技术爱好者，喜欢上传关于全栈开发的教程和他的旅行 Vlog。

Coding Garden 一个开放、互动和引人入胜的社区，任何编码人员，从初学者到老手，都可以在这里学习和成长。无论是教程、问答环节、算法问题解决还是完整的应用程序设计和构建，总有新东西要学。

JREAM 如果你作为初学者开始学习Web开发，应该在YouTube上查看JREAM，该频道包含许多开发教程和一些 Adobe 视频，是学习PHP、MySQL、Linux 等流行Web技术的理想场所。

Neil Rowe 从Bootstrap到Laravel基本知识，Neil Rowe的频道是学习高级Web开发技巧和窍门的顶级YouTube资源。他上传了简单易懂的网络教程，帮助初学者在一个地方找到所有开发方法。

Joshua Fluke 你会在Joshua的YouTube频道上找到一些抽象的视频，但他最受关注的流媒体站的真正内容是网络开发和设计视频。他喜欢制作面向社会的开发视频。

Jesse Showalter Jesse的官方YouTube频道对于想要轻松学习网络和数字设计的网络工匠来说是一个不错的地方。你会在频道上找到一些方便的UI/UX设计教程，这些教程有助于提升你的技能。

Chris Coyier 对于所有想要掌握前端脚本的CSS爱好者，向他们推荐这个频道，因为它有大量涵盖多个CSS主题的视频。它还为初学者提供了一些高级WordPress教程和一些Photoshop视频指南。

Chris Sean Chris是一名专业的Web开发人员，喜欢上传有关热门开发主题的视频。大多数视频都描述了他作为Web开发人员的经历，提供了关于他的生活以及他作为开发人员喜欢做的事情的见解。

Tuts+ Web Design 对于想要快速开始网页设计的新手来说，Tuts+ 是一个完美的流媒体频道。它有几个关于构建响应式布局、使用Twitter引导程序、方便的CSS技巧等的视频。

Naveed Ziarab 此频道提供了有关主要后端框架和语言的详细教程，包括Laravel、CodeIgniter、Java、C#/C++ 等。对于绝对的初学者，它有一个单独的 OOP编程播放列表，有助于建立基本的开发基础。除此之外，它还有一个全面的数据库和Twitter引导教程。

Joseph Smith 作为一名经验丰富的前端开发人员，Joseph喜欢上传有关最新前端技术的视频，例如Node、Next.js等。对于初学者，该频道有一些基本的 PHP、HTML和CSS视频，可帮助初露头角的开发人员轻松获得实践经验。

Eleftheria Batsou 最好的学习方法是教别人，这个频道试图通过小而有趣的教程来解释我对Web开发和Web设计的想法。

FreeCodeCamp 免费学习编码。几乎所有编码语言的教程数量几乎无限。

LearnCode.academy 100%免费的Web开发教程、网站设计教程等。包括但不限于：HTML、CSS、JavaScript、CSS 布局、响应式设计、React.js、Node.js、Angular.js、Docker、Dev Ops、服务器管理、部署策略。

CSS tricks 这是CSS-Tricks的官方YouTube频道，一个由Chris Coyier策划的网页设计社区。| 官网

Adam Khoury 像专业人士一样学习编码和设计。掌握JavaScript、PHP、SQL、HTML、CSS、ActionScript等开发技术。

Fun Fun Function Mattias Petter Johansson，简称mpj。他做了大约十年的全职程序员。他曾为Absolut Vodka、Blackberry、Spotify工作过。

Derek Banas 他根据你的要求制作教程，涵盖你可以想象的任何主题。每天都会上传新视频。他目前正在教授一系列关于数据科学和机器学习的课程。

Clément Mihailescu 他是前谷歌软件工程师，前Facebook软件工程师，以及AlgoExpert ( algoexpert.io ) 的CEO和联合创始人)，一个帮助软件工程师准备编码面试的网站。这个YouTube频道是他关于科技、创业和随机杂项的所有想法的出路！

DevTips 是每周一次的节目，面向希望获得灵感和学习编程的你。由 Andrew 主持 - 一位长期的社区成员、bug破坏者和设计师。让我们一起探索代码并一路学习编程

1stWebDesigner 主要强调网页设计，你会在这个频道上找到一些非常方便的教程，这些教程精确地展示了网页设计的艺术细节。该频道也是WordPress开发教程的一个很好的来源，让初学者知道如何快速开始使用WP。
