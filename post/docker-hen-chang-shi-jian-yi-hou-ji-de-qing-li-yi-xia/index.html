Docker 很长时间以后记得清理一下

2024-12-18 1 min read # docker

系统中用 Docker 很长时间以后记得清理一下：

docker system prune -a --volumes

-a:

  • 所有停止的容器
  • 所有未使用的网络
  • 所有 Dangling 和未使用的镜像
  • 所有构建缓存

--volumes：同时清理未使用的数据卷
