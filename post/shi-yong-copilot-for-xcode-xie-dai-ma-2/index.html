1.4 Github Copilot 免费到期了怎么办？

Github Copilot 免费到期了，有三个免费的替代：

  1. Codeium：不能更换模型，但可以免费无限使用。
  2. Tabnine：开源，支持本地模型。
  3. https://Continue.dev 开源，支持本地和API模型。但是Tab补全推荐使用sft后的本地模型。
