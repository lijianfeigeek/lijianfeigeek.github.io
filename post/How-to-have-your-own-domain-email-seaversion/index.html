如何拥有自己的域名邮箱【海外版】

2023-04-01 2 min read # 原创

之前写过一篇 如何拥有自己的域名邮箱 算是国内版

如何拥有自己的域名邮箱 | lijianfei.com

今天我们来聊聊海外版。

使用 Cloudflare 电子邮件路由 可以将个人域名的邮箱转发至其它邮箱，本文我们通过 Gmail 做到使用个人域名收发邮件。

Cloudflare 接收邮件

第一步，请确保已在 Cloudflare 上添加了用于收发邮件的站点（没有请看 Cloudflare添加站点 ），然后点击 电子邮件 -> 开始使用

第二步，在 自定义地址 栏输入邮箱前缀，目标地址 栏输入 Gmail 地址，点击 创建并继续：

第三步，如果是新地址则会收到 验证电子邮件路由地址 邮件，点击 验证电子邮箱地址，验证成功后会提示 启用电子邮件路由，点击 添加记录并启用：

第四步，当显示 配置摘要 时，完成！

Gmail 发送邮件

第一步，接下来使用 应用专用密码 为 Gmail 生成一个密码。
在 选择应用 列中选择 其它（自定义名称），填一个好记的名字（例如邮箱前缀），点击 生成：

第二步，保存黄框中的 应用专用密码，点击 完成：

第三步，打开 Gmail，点击 设置 -> 查看所有设置 -> 账号和导入 -> 用这个地址发送邮件： -> 添加其他电子邮件地址：

第四步，此时会弹出一个新窗口，在 名称： 行输入邮箱的名称（会对外展示，请慎重填写），在 电子邮件地址： 行输入 Cloudflare 中添加的邮箱地址（请提前确认可以接收邮件），点击下一步，如下图所示：

第五步，接下来会提示 通过SMTP服务器发送邮件，在 SMTP服务器： 行输入smtp.gmail.com，用户名 行输入 Gmail 的用户名（邮箱不带 @gmail.com），密码 行输入之前保存的 应用专用密码，最后点击 添加帐号 »：

第六步，如果配置无误，Gmail 会受到一封邮件，填入邮件的确认代码并点击 验证：

大功告成。

      • Cloudflare 接收邮件
      • Gmail 发送邮件
