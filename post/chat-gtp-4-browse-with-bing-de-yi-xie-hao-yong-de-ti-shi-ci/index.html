Chat GTP-4 Browse with Bing 的一些好用的提示词

2023-10-04 2 min read # AI

ChatGPT 现在可以再次访问互联网，这使得它能够完成以前不可能完成的事情。

1. 检测趋势

由于它连接到互联网，ChatGPT可以成为您最好的商业伙伴。

提示→ 提供2023年第一季度最新电商趋势和消费者行为模式的简短分析

2. 从最新来源编写内容

ChatGPT可以搜索最新的信息来编写最新的内容。

提示 → 分析近几个月来西方和金砖国家领导人的受欢迎程度，并写一篇简短的博文

3. 股市分析师

利用实时数据让 ChatGPT 分析公司业绩。

提示 → 根据科技公司的最新财务业绩，未来几周它们的股价将如何变化

4. 根据即将发生的事件制定计划

使用互联网访问让 ChatGPT 根据天气或时事调整其结果。

提示 → 查看法国阿尔卑斯山未来几天的天气预报，并告诉我相应的徒步包里应该装些什么

5. 学术研究

节省分析最新学术文献的时间。

提示 → 总结上周《自然》杂志发表的有关系外行星破坏的最新研究的主要发现。在markdown中给出答案

6. 获取反馈

实时了解人们对某件事的看法，而无需逐页搜索。

提示 → Reddit 上的人们如何看待最新的图像生成模型 Dall-E 3

7. 使用最新文档进行编码

ChatGPT 会搜索最新的信息来更好地编码。

提示 → 使用最新的可用文档对 Chrome 扩展的清单文件的结构进行编码

8. 引用可靠来源

可以找出 ChatGPT 使用哪些网络源来检查它们。

提示 → 在互联网上搜索有关柏林墙的最可靠消息来源。然后解释它是如何受到保护的，并引用你的消息来源
